Haftalık Ders Planı Web Prototipi

Kullanım:
1. index.html dosyasını çift tıklayarak tarayıcıda açın.
2. Sol menüden Genel Bakış, Haftalık Plan, Öğretmenler, Ödevler ve Rol Sistemi ekranlarını inceleyin.
3. Sınıf seçimi üst sağdan yapılır.
4. PDF/Yazdır butonu tarayıcı yazdırma ekranını açar.

Dosyalar:
- index.html: Ana arayüz
- style.css: Tasarım
- app.js: Ekran davranışları
- data.js: Excel’den çıkarılan örnek veri

Not:
Bu bir statik prototiptir. Sonraki aşamada veri tabanı, giriş sistemi ve yönetim panelleri eklenebilir.
